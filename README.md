# VScode-EasyX-
>——在VScode上导入C++的第三方函数库EasyX，配置相关的.json文件。

#### 1.下载对于MinGW的EasyX库
>[跳转到下载页面](https://easyx.cn/t/download)

#### 2.添加库文件和头文件到mingw64文件目录
>对已下载的include和lib64/32文件夹自定义重命名
>
>再分别放入mingw64文件表目录的include和lib文件夹
>
>其中include文件夹内为.h头文件，lib文件夹内为.a库文件
